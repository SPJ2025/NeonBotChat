# NeonBotChat v26.7.6 更新日志

## 🆕 新功能

- **文件大小限制**：富媒体发送前检查文件大小
  - 超过软限制弹窗确认（query 图标），可继续发送
  - 超过 200MB 硬限制弹窗阻止（error 图标），无法发送
  - 软限制：图片 20MB / 视频 30MB / 语音 200MB / 文件 200MB
- **QQ API 错误检测**：发送消息后检查 API 返回的错误码，失败时前端显示具体错误信息
- **重发显示气泡**：失败后点击重发时显示 pending 气泡，不再是空白消失
- **error.svg 图标**：硬限制提示使用独立错误图标

## 🔧 改进

- **透明度滑块**：min 从 5 改为 0，滑块位置与视觉起点对齐；代码锁死最低 5% 透明度
- **自定义弹窗增强**：支持动态切换确认按钮颜色、支持隐藏取消按钮
- **登录页 Bot 图标**：添加 `filter: invert(1)` 适配深色背景
- **引用框 Bot 图标**：使用 `fmtBotMarker` 渲染 bot.svg
- **图标不反色列表**：新增 `query.svg`、`warning.svg`、`error.svg`
- **Range 滑块统一样式**：跨浏览器自定义 track 和 thumb 样式
- **删除操作先关菜单**：无延迟感

## 🐛 修复

- 修复发送失败时仍显示「发送成功」
- 修复重发时气泡消失
- 修复透明度滑块最小值为 5 导致起点有间隙
- 修复登录页 Bot 图标在深色背景下不可见
- 修复引用框 Bot 名称未使用 bot.svg 图标
- 修复 `send_group_msg` 和 `upload_group_media_by_url` 错误未检测
- 修复 `error.svg` 不反色
