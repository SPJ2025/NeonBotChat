# 🤖 NeonBotChat

基于 QQ 官方 Bot API 的 WebUI 机器人接管面板，支持真人接管 Bot 进行群消息收发。

## ✨ 功能

### 消息收发
- **实时消息**：WebSocket 推送，秒级同步群消息
- **手动接管**：通过 WebUI 直接以 Bot 身份发送消息
- **富媒体发送**：图片/视频/语音/文件通过图床一键发送，含大小限制和确认提示
- **富媒体接收**：图片 / 视频 / 语音 / 文件自动渲染播放
- **富媒体转发**：图片 / 视频 / 语音转发为真实媒体消息
- **Markdown 消息**：勾选 MD 发送，支持标题、粗体、斜体、代码、LaTeX 公式渲染
- **乐观更新**：发送消息即时显示气泡（含媒体预览），失败可重发
- **引用回复 UI**：毛玻璃引用框，显示发送人和内容，Bot 图标自动适配
- **右键菜单**：复制（文字 / 图片）、下载视频、转发、收藏、撤回、删除、多选
- **语音转文字**：语音消息下方显示 QQ 语音识别文本

### 消息管理
- **消息历史**：本地 SQLite 持久化，自动 5 秒刷新
- **撤回消息**：调用 QQ API 撤回，文字消息支持「重新编辑」
- **删除消息**：本地删除单条或批量，不影响 QQ 端
- **多选模式**：右键进入多选，批量删除消息
- **清空记录**：支持清空单个群聊或全部聊天记录

### 会话管理
- **聊天列表**：自动识别群聊，预览显示「发送人: 内容」
- **备注名**：给群聊设置自定义备注名
- **添加群聊**：手动输入 group_openid 添加会话

### 用户识别
- **Bot 标识**：Bot 用户自动标注 🤖
- **角色徽章**：群主（黄色发光）/ 管理员（蓝色发光）毛玻璃徽章
- **角色颜色**：群主名黄色 / 管理员名蓝色 / 普通用户白色
- **@提及转昵称**：`<@OpenID>` 自动替换为 @昵称
- **用户头像**：消息旁显示，点击可查看大图

### 收藏系统
- 右键收藏任意消息（文字 / 图片 / 视频 / 语音）
- 收藏列表支持图片预览、视频播放、语音播放、转文字显示

### UI / 个性化
- **SVG 图标**：全站 Lucide 风格 SVG，CSS 自动适配深浅色
- **自定义弹窗**：毛玻璃确认弹窗，替代浏览器原生对话框
- **深浅模式**：深色 / 浅色主题一键切换，客户端独立
- **自定义背景**：上传图片作为聊天背景，填充整个页面
- **Bot 头像 / 简介**：服务器端存储，跨设备共享
- **透明度调节**：背景透明度 5%~100% 可调
- **毛玻璃效果**：侧边栏、顶栏、输入栏、气泡、按钮、设置导航栏均支持毛玻璃
- **移动端适配**：≤768px 自动全屏聊天，左上角返回按钮
- **消息时间**：悬停 1 秒后气泡外显示时间，不占空间

### 设置面板
- **通用**：外观（深浅模式）+ 聊天背景（上传 / 透明度 / 清除）+ 清空全部聊天记录
- **账号与安全**：Bot 头像 / 简介 + 退出登录
- **状态**：系统信息、CPU / 内存实时监控，进度条平滑动画
- **关于**：版本信息 + README 链接

### 安全
- **密码保护**：`webui-password` 启用后显示登录页验证
- **图床鉴权**：上传需 Bearer Token

### 配置
- 端口可配置（`webui-port`），默认 8080
- 外网访问开关（`enable-public`）
- 图床服务器独立配置（`file_server/config.yaml`）

## 🚀 快速开始

### 环境要求

- Python 3.10+
- Windows / Linux / macOS

### 安装

```bash
pip install qq-botpy aiohttp fastapi uvicorn psutil
```

### 配置

复制 `config.example.yaml` 为 `config.yaml` 并填入真实值：

```yaml
appid: "你的AppID"
secret: "你的AppSecret"
webui-port: 36336          # WebUI 端口，默认 8080
enable-public: false       # true 绑定 0.0.0.0 允许外网访问
webui-password: ""         # 访问密码，留空不启用

# 图床服务器连接（可选，用于富媒体发送）
file-server:
  public-url: "http://你的域名:36337"
  token: "你的Token"
```

### 启动

```bash
# 主程序
python init.py

# 图床服务器（可选，用于发送富媒体）
python file_server/file_server.py
```

浏览器打开 `http://127.0.0.1:36336`。

## 📁 项目结构

```
NeonBotChat/
├── init.py              # 主入口，启动 Bot + Web 服务
├── web_server.py        # FastAPI + WebSocket 后端
├── database.py          # SQLite 存储 + 消息去重
├── PatchMsg.py          # 补丁：群消息解析 + 附件提取 + 角色识别
├── PatchActiveMsg.py    # 补丁：主动发送 / 撤回 / URL 上传
├── PatchUserInfo.py     # 补丁：获取用户昵称 / 头像
├── config.example.yaml  # 配置文件示例
├── config.yaml          # 实际配置（含密钥，不提交 Git）
├── .gitignore           # Git 忽略规则
├── README.md            # 项目文档
├── templates/
│   └── index.html       # WebUI 前端（零框架依赖）
└── file_server/         # 图床服务器（独立项目）
    ├── file_server.py
    ├── config.yaml
    └── config.example.yaml
```

## 🛠 技术栈

- **后端**：Python FastAPI + WebSocket
- **前端**：原生 HTML/CSS/JS（零框架依赖）
- **存储**：SQLite
- **Bot SDK**：qq-botpy 1.2.1

## ⚠️ 注意事项

- `config.yaml` 中的 AppSecret 是敏感信息，请勿泄露或提交到公开仓库
- qq-botpy 官方 SDK 已长期未更新，本项目通过 Patch 模块补齐缺失功能
- 图片 / 视频 / 语音依赖 QQ 多媒体 CDN，链接可能有时效性
- 撤回消息有时限，超时无法撤回
- 语音消息使用 `voice_wav_url`（WAV 格式），非原始 AMR
- 图床服务器为独立项目，需单独启动，可与主程序部署在同一机器
- 富媒体发送需要图床服务器 + QQ API 支持，QQ 服务器需能访问图床公网 URL
