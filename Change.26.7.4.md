# NeonBotChat v26.7.4 更新日志

## 🆕 新功能

- **图床 / 文件服务器**：独立的 `file_server/` 项目，支持图片/视频/语音上传
  - SHA256 去重：相同文件只存一份
  - 自动缓存清理：超限时删除最旧文件
  - 独立配置：`file_server/config.yaml`
  - 完善日志：上传/去重/清理均有日志
- **富媒体发送**：输入框 ＋ → 图片 / 视频 / 语音
  - 上传到图床 → 获取公网链接 → 通过 QQ 富媒体 API 发送
  - 发送中显示本地预览气泡（图片/视频/音频播放器）
  - 失败可重试
- **富媒体转发**：图片 / 视频 / 语音右键转发为真实媒体消息（非链接）
  - 先关窗口再处理，无卡顿感
  - 图床直链直接使用，QQ CDN 先上图床再发送
- **语音转文字**：语音消息下方显示 `asr_refer_text`，收藏列表也支持
- **服务器端设置**：Bot 头像 / 简介存储在 `bot_settings.json`，跨设备共享
- **消息时间外置**：悬停 1 秒后气泡外侧显示，不撑开气泡
- **图片加载占位**：等比例灰色占位框，加载完淡入
- **媒体过期提示**：「图片/视频已过期或已被清理」
- **表情识别**：`faceType=1` → `[表情符号]`，`faceType=6` → 贴纸小图（120×120）
- **Markdown 渲染**：h1-h6 标题、粗体、斜体、代码、LaTeX 公式
- **文件消息**：根据后缀显示对应图标，点击下载

## 🔧 改进

- **毛玻璃按钮**：全部设置按钮改为无边框毛玻璃样式
- **浅色模式优先**：主题切换按钮浅色在前
- **徽章发光**：群主/管理员标签添加同色系 `box-shadow`
- **浅色模式徽章加深**：更易读
- **撤回消息不弹右键菜单**
- **富媒体撤回不显示「重新编辑」**
- **多选模式屏蔽文件/图片/视频点击**
- **右键菜单底部自动翻转**
- **发送按钮修复**（事件参数误传）
- **未读计数**：自己发的消息不加红点
- **移动端适配**：≤768px 全屏聊天 + 返回按钮
- **图床独立日志**：完整的 logging 记录
- **设置独立化**：主程序和图床各自读自己的 `config.yaml`
